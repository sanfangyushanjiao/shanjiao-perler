# 快速启动指南

## 环境要求

- Node.js 18+
- npm 或 yarn

## 安装步骤

1. 进入项目目录：
```bash
cd C:\Users\ASUS\perler-beads-project\my-perler-app
```

2. 安装依赖（如果还没安装）：
```bash
npm install
```

## 开发模式

启动开发服务器：
```bash
npm run dev
```

然后在浏览器中打开显示的地址（通常是 http://localhost:5173）

## 生产构建

构建生产版本：
```bash
npm run build
```

预览生产版本：
```bash
npm run preview
```

## 功能测试

### 测试图片上传
1. 准备一张测试图片（建议尺寸：500x500 左右）
2. 拖拽或点击上传区域上传图片
3. 等待处理完成

### 测试参数调整
1. 调整网格尺寸滑块（20-200）
2. 切换像素化模式（卡通/真实）
3. 切换品牌选择
4. 观察预览和统计的变化

### 测试导出功能
1. 处理完图片后
2. 点击"导出图纸"按钮
3. 检查下载的 PNG 文件

## 常见问题

### Q: 图片处理很慢怎么办？
A: 尝试降低网格尺寸，或使用较小的图片。

### Q: 颜色不准确怎么办？
A: 尝试切换像素化模式，或调整图片的对比度和亮度。

### Q: 导出的图纸太大/太小？
A: 目前导出尺寸是固定的，后续版本会添加自定义尺寸功能。
