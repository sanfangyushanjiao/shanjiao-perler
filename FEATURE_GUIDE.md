# 拼豆图纸生成工具 - 功能体验指南

## 访问地址
**开发服务器：** http://localhost:5173

## 🎯 快速体验所有新功能

### 第一步：上传图片体验基础功能

1. **打开浏览器访问** http://localhost:5173
2. **准备一张图片**（建议使用简单的图片，如 logo、图标等）
3. **上传图片**：
   - 方式1：点击上传区域选择图片
   - 方式2：直接拖拽图片到上传区域

---

### 第二步：体验图像处理功能

#### 2.1 调整网格尺寸
- 在右侧"控制面板"中找到"网格尺寸"
- 输入数值（建议：30-50）
- 点击"应用参数"按钮
- **效果**：图片会被转换为指定大小的像素网格

#### 2.2 颜色数量控制
- 在"颜色数量控制"输入框输入数值（建议：10-30）
- 点击"应用参数"
- **效果**：相似的颜色会被合并，减少颜色种类

#### 2.3 去除背景
- 点击"去除背景"按钮
- **效果**：图片外围的背景会被标记为灰色

#### 2.4 切换品牌
- 在"拼豆品牌"下拉框选择不同品牌
- **效果**：色号会根据品牌变化

---

### 第三步：体验手动编辑功能 ⭐

#### 3.1 进入编辑模式
- 点击右侧"编辑工具"卡片中的"进入编辑"按钮
- **效果**：工具栏会展开显示编辑工具

#### 3.2 使用画笔工具
1. 点击"画笔"工具（🖌️图标）
2. 在下方"调色盘"中选择一个颜色
3. 在画布上点击或拖拽绘制
4. **效果**：可以手动修改像素颜色

#### 3.3 使用橡皮擦
1. 点击"橡皮擦"工具（🧹图标）
2. 在画布上点击或拖拽
3. **效果**：擦除的区域变为白色

#### 3.4 使用区域擦除
1. 点击"区域擦除"工具（🗑️图标）
2. 点击画布上的某个颜色区域
3. **效果**：相同颜色的连续区域会被全部擦除

#### 3.5 批量替换颜色
1. 点击"批量替换"工具（🔄图标）
2. 点击画布上要替换的颜色（步骤1）
3. 在调色盘中选择目标颜色（步骤2）
4. **效果**：所有相同颜色的像素会被替换

#### 3.6 撤销/重做
- 点击"↶ 撤销"按钮：撤销上一步操作
- 点击"↷ 重做"按钮：恢复撤销的操作
- **支持无限次撤销/重做**

---

### 第四步：体验画布操作功能

#### 4.1 缩放画布
**PC 端：**
- 按住 Ctrl（或 Cmd）+ 滚动鼠标滚轮
- 或点击"缩小"/"放大"按钮

**移动端（如果有触摸屏）：**
- 双指捏合：缩小
- 双指展开：放大

#### 4.2 移动画布
**PC 端：**
- 在非编辑模式下，按住鼠标左键拖拽

**移动端：**
- 单指拖拽移动画布

#### 4.3 重置视图
- 点击"重置"按钮
- **效果**：画布恢复到初始缩放和位置

---

### 第五步：体验导出功能 ⭐

#### 5.1 导出 PNG 图纸
1. 点击左下方"导出图纸"按钮
2. **效果**：
   - 下载一张 PNG 图片
   - 包含完整的像素网格和色号
   - 底部有颜色统计表
   - 显示绿色成功动画 ✅

#### 5.2 导出 CSV 文件
1. 点击"导出 CSV"按钮
2. **效果**：
   - 下载一个 CSV 文件
   - 包含所有像素的坐标、颜色、色号
   - 可以在其他设备上重新导入

## 🎨 推荐体验流程

### 流程1：从图片到图纸（完整流程）
```
上传图片 → 调整网格尺寸 → 颜色合并 → 去除背景
→ 手动编辑 → 导出图纸
```

### 流程2：CSV 跨设备工作流
```
PC端：上传图片 → 编辑 → 导出CSV
→ 传输到手机
手机端：上传CSV → 继续编辑 → 导出图纸
```

### 流程3：精细化编辑
```
上传图片 → 自动去除杂色 → 进入编辑模式
→ 使用画笔修饰 → 批量替换颜色 → 导出
```

## 🐛 常见问题

### Q1: 为什么我访问的是 5174 端口？
**A:** Vite 默认端口是 5173，如果被占用会自动使用 5174。请访问 **http://localhost:5173**

### Q2: CSV 文件有什么用？
**A:** CSV 文件可以：
- 保存图纸数据（比图片文件小）
- 跨设备编辑（PC、手机、平板）
- 与他人分享设计
- 批量处理数据

### Q3: 如何在移动端测试触摸功能？
**A:**
- 方式1：使用真实的触摸设备访问
- 方式2：Chrome DevTools 模拟触摸（F12 → 设备模式）
- 方式3：使用触摸屏笔记本

### Q4: 导出的图纸包含什么？
**A:** PNG 图纸包含：
- 完整的像素网格（带色号）
- 底部的颜色统计表
- 品牌信息
- 总珠子数

### Q5: 如何获得最佳效果？
**A:**
- 使用清晰、对比度高的图片
- 从小网格开始（30-50）
- 逐步调整颜色合并阈值
- 使用"去除背景"简化图案
