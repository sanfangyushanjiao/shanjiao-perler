## 📋 项目概述

拼豆图纸生成工具是一个现代化的 Web 应用，帮助用户将图片转换为拼豆图纸。经过三个版本的迭代，现已成为功能完善、体验优秀的生产级应用。

## 🎨 设计规范

### 颜色系统
```css
primary: #FF6B9D   /* 粉红 - 主色调 */
secondary: #4ECDC4 /* 青绿 - 次要色 */
accent: #FFE66D    /* 黄色 - 强调色 */
```

### 间距系统
```css
card: 24px    /* 卡片内边距 */
section: 16px /* 区块间距 */
```

### 圆角系统
```css
card: 16px    /* 卡片圆角 */
button: 8px   /* 按钮圆角 */
```

### 阴影系统
```css
card: 0 4px 6px -1px rgba(0, 0, 0, 0.1)
card-hover: 0 10px 15px -3px rgba(0, 0, 0, 0.1)
primary: 0 4px 14px 0 rgba(255, 107, 157, 0.39)
```
