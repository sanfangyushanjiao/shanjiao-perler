## 🎯 方案一：拼多多电子卡券（推荐，像截图）

### 第一步：准备激活码文件

您已有的 CSV 文件需要转换格式：

#### 打开 activation-codes-24h.csv

文件内容类似：
```csv
code,type,price,status,created_at
SJ-A1B2-C3D4-E5F6,24h,1.80,unused,2026-03-05
SJ-B2C3-D4E5-F6A7,24h,1.80,unused,2026-03-05
...
```

#### 提取激活码列（只要第一列）

**方法一：用 Excel**
1. 用 Excel 打开 activation-codes-24h.csv
2. 选中第一列（code 列）
3. 复制
4. 新建一个 TXT 文件
5. 粘贴
6. 保存为 `拼多多-24小时激活码.txt`

**方法二：用记事本**
1. 用记事本打开 activation-codes-24h.csv
2. 手动删除第一行表头和其他列
3. 只保留激活码
4. 保存为 `拼多多-24小时激活码.txt`

**方法三：用我们的转换脚本（最快）**
```bash
# 您已经有这个脚本了！
cd C:\Users\ASUS\shanjiao-perler
npm run convert:dujiaoka
```
这会生成：
- dujiaoka-codes-24h.txt ✅
- dujiaoka-codes-7d.txt ✅
- dujiaoka-codes-lifetime.txt ✅
