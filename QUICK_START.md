# 快速开始 - 5分钟上手指南

## 🚀 启动应用

```bash
# 1. 进入项目目录
cd perler-beads-project/my-perler-app

# 2. 启动开发服务器
npm run dev

# 3. 打开浏览器访问
# http://localhost:5173
```

---

## 🎯 3个核心功能快速体验

### 1️⃣ 图片转图纸（1分钟）

```
1. 准备一张简单的图片（如 logo）
2. 拖拽到上传区域
3. 等待处理完成
4. 点击"导出图纸"按钮
✅ 完成！你得到了一张拼豆图纸
```

### 2️⃣ 手动编辑（2分钟）

```
1. 上传图片后，点击"进入编辑"
2. 选择"画笔"工具（🖌️）
3. 在调色盘选择颜色
4. 在画布上点击绘制
5. 点击"↶ 撤销"可以撤销
✅ 完成！你可以手动修改图纸
```

### 3️⃣ CSV 保存和分享（2分钟）

```
1. 生成图纸后，点击"导出 CSV"
2. 点击"重新开始"
3. 拖拽刚才的 CSV 文件到上传区域
4. 图纸完全恢复
✅ 完成！你可以保存和分享图纸数据
```

## 📱 移动端体验

如果你有触摸设备：

```
1. 用手机浏览器访问 http://你的电脑IP:5173
2. 上传图片
3. 双指捏合/展开 → 缩放画布
4. 单指拖拽 → 移动画布
5. 进入编辑模式，用手指绘制
✅ 完成！移动端体验流畅
```

## 🎨 推荐工作流

### 简单图案（5分钟）
```
上传图片 → 调整网格尺寸(30) → 导出图纸
```

### 精细图案（10分钟）
```
上传图片 → 调整网格尺寸(50) → 颜色合并(20)
→ 去除背景 → 手动编辑 → 导出图纸
```

### 跨设备编辑（15分钟）
```
PC: 上传图片 → 初步编辑 → 导出CSV
手机: 上传CSV → 继续编辑 → 导出图纸
```

## ❓ 常见问题

**Q: 端口是 5173 还是 5174？**
A: 默认是 5173，如果被占用会自动使用 5174

**Q: CSV 文件有什么用？**
A: 保存图纸数据，可以跨设备编辑和分享

**Q: 如何获得最佳效果？**
A: 使用清晰图片，从小网格开始（30-50）

**Q: 支持哪些图片格式？**
A: JPG、PNG、GIF 等常见格式

**Q: 可以在手机上使用吗？**
A: 可以！完美支持触摸操作
