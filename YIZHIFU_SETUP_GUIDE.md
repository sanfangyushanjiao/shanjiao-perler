## 1. 注册与设置

### 1.1 注册账号

**推荐平台**：

由于"易支付发卡网"可能有多个类似平台，以下是几个主流的自动发卡平台：

#### 选项 A：卡易售 ⭐ 推荐
- 官网：https://www.kayisou.com
- 优点：界面友好，支持多种支付方式
- 手续费：0.38%

#### 选项 B：发卡星
- 官网：https://www.fakastar.com
- 优点：功能强大，自定义程度高
- 手续费：0.3%

#### 选项 C：彩虹发卡
- 官网：自建开源系统
- 优点：免费，完全掌控
- 手续费：0（仅支付通道费用）

**本指南以"卡易售"为例，其他平台操作类似**

---

### 1.2 注册步骤

1. **访问平台官网**
   - 点击"免费注册"或"立即开店"

2. **填写注册信息**
   ```
   用户名: 自定义（建议：shanjiao-perler）
   密码: 设置强密码
   邮箱: 你的常用邮箱
   手机号: 接收验证码
   ```

3. **邮箱验证**
   - 查收验证邮件
   - 点击链接完成验证

4. **登录后台**
   - 使用账号密码登录管理后台

---

### 1.3 完成实名认证（部分平台需要）

进入后台 → 账户设置 → 实名认证

**需要准备**：
- 身份证正反面照片
- 手持身份证照片
- 银行卡信息（用于提现）

**审核时间**：通常 1-24 小时

---

### 1.4 配置支付方式

进入后台 → 支付设置

**推荐配置**：

| 支付方式 | 建议 | 原因 |
|---------|------|------|
| 支付宝 | ✅ 必选 | 用户最多 |
| 微信支付 | ✅ 必选 | 用户习惯 |
| QQ钱包 | ⚪ 可选 | 年轻用户 |
| 银联 | ⚪ 可选 | 大额支付 |

**配置方法**（以支付宝为例）：
1. 点击"添加支付宝"
2. 填写支付宝账号
3. 上传收款二维码（个人版）
4. 或对接官方支付宝（需企业认证）

## 3. 上传激活码

### 3.1 准备激活码CSV文件

在本地生成激活码：

```bash
# 进入项目目录
cd shanjiao-perler

# 创建 .env 文件（如果还没有）
cp .env.example .env

# 编辑 .env，填入 Supabase 配置
# SUPABASE_URL=https://xxx.supabase.co
# SUPABASE_SERVICE_KEY=eyJhbGc...

# 生成激活码
npm run gen:codes:24h      # 生成200个24小时码
npm run gen:codes:7d       # 生成100个7天码
npm run gen:codes:lifetime # 生成50个永久码
```

执行后会生成三个CSV文件：
- `activation-codes-24h-batch-xxx.csv`
- `activation-codes-7d-batch-xxx.csv`
- `activation-codes-lifetime-batch-xxx.csv`

### 3.2 上传到发卡平台

#### 方法一：通过后台批量导入

1. **进入商品详情**
   - 后台 → 商品管理 → 选择"24小时体验版"

2. **点击"卡密管理"**

3. **选择"批量导入"**

4. **上传CSV文件**
   - 选择 `activation-codes-24h-batch-xxx.csv`
   - 确认字段映射：
     ```
     卡密字段 → code 列
     价格字段 → price 列（可选）
     类型字段 → type 列（可选）
     ```

5. **确认导入**
   - 检查预览数据
   - 点击"开始导入"
   - 等待导入完成

6. **重复以上步骤**
   - 为"七天版"上传 `activation-codes-7d-batch-xxx.csv`
   - 为"永久版"上传 `activation-codes-lifetime-batch-xxx.csv`

#### 方法二：手动粘贴（小批量）

如果平台支持直接粘贴：

1. 打开CSV文件
2. 复制所有激活码（仅 code 列）
3. 粘贴到发卡平台的"批量添加"文本框
4. 每行一个激活码
5. 提交

---

## 8. 故障处理

### 8.1 常见问题

#### 问题1：发卡平台无法自动发货

**可能原因**：
- 库存不足
- 商品设置错误
- 支付未到账

**解决方案**：
1. 检查商品库存
2. 检查商品设置（确认开启自动发货）
3. 查看订单状态
4. 手动补发激活码

#### 问题2：买家收不到激活码

**解决方案**：
1. 登录发卡平台后台
2. 订单管理 → 找到对应订单
3. 查看发货状态
4. 如未发货，点击"手动发货"
5. 或直接私信买家发送激活码

#### 问题3：激活码重复发送

**原因**：系统bug或重复操作

**解决方案**：
- 记录已发送的激活码
- 如确实重复，标记该激活码为已使用
- 后续订单跳过该激活码

---

### 8.2 应急预案

**情况1：数据库激活码用完**

```bash
# 紧急生成激活码
tsx scripts/generate-codes.ts --type lifetime --count 20

# 立即上传到发卡平台
```

**情况2：网站无法访问**

- 检查 Vercel 部署状态
- 查看域名DNS设置
- 临时提供备用访问方式

**情况3：大量差评**

1. 立即暂停销售
2. 逐一联系买家解决问题
3. 提供补偿（延长使用期或退款）
4. 修复问题后再恢复销售
