# 更新日志

1. **参数输入方式优化**

- 更清晰的操作流程

- 更符合用户预期

## v0.4.0 - 2026-03-01

### 新增功能 ✨

1. **去除背景功能**
   - 自动识别并移除白色/浅色背景
   - 使用洪水填充算法从边界开始标记外部区域
   - 预览时外部区域显示为浅灰色
   - 统计和导出时自动排除背景区域
   - 可通过开关控制是否启用

2. **画布缩放和平移**
   - 画布居中显示，自适应缩放以完整显示
   - 缩放控制按钮（放大/缩小/重置）
   - 鼠标滚轮缩放支持
   - 拖拽平移功能
   - 实时显示缩放比例

### 优化改进 🔧

1. **参数控制优化**
   - "横轴切割数量" 改名为 "网格尺寸"
   - "颜色合并阈值" 改名为 "颜色数量控制"
   - 改回滑块控制，更直观
   - 保留"应用参数"按钮确认机制

2. **预览体验提升**
   - 画布完整显示，不再需要滚动
   - 居中对齐，视觉效果更好
   - 支持自由缩放查看细节
   - 流畅的交互体验

### 技术改进 💻

- 新增 `backgroundRemoval.ts` 模块
- 实现洪水填充算法标记外部区域
- 优化画布渲染逻辑
- 改进状态管理

## v0.2.0 - 2026-03-01

### 新增功能 ✨

1. **颜色合并控制**
   - 添加颜色合并阈值滑块（0-100）
   - 使用 BFS 算法自动合并相似颜色
   - 实时调整，立即生效
   - 可有效减少颜色种类，简化拼豆制作

2. **增强导出功能**
   - 导出图像现在包含完整的颜色统计信息
   - 布局：标题 + 像素图 + 统计表
   - 统计表采用 3 列布局，清晰展示色号和数量
   - 渐变色标题，更美观

### 优化改进 🔧

1. **预览画布优化**
   - 动态计算单元格大小，避免失真
   - 最大高度限制为 70vh，防止占用过多空间
   - 字体大小根据单元格大小自适应
   - 添加边框，视觉效果更清晰

2. **简化操作**
   - 移除像素化模式选项，固定使用真实模式（平均色）
   - 减少用户选择，降低使用门槛
   - 界面更简洁

### 技术改进 💻

- 新增 `colorMerge.ts` 模块，实现 BFS 颜色合并算法
- 优化状态管理，分离原始网格和处理后网格
- 改进导出函数，支持传入颜色统计数据
- 代码结构更清晰，易于维护

### 问题修复 🐛

- 修复预览画布失真问题
- 修复导出图像缺少统计信息的问题
