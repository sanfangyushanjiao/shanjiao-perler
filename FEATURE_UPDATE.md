# 功能更新说明

## 新增功能

### 1. 画布自适应显示 ✅

**功能描述**：
- 无论网格尺寸多大（10x10 或 300x300），画布都会自动缩放以完整显示在容器中
- 用户可以通过缩放按钮或 Ctrl/Cmd + 滚轮自由调整查看比例
- 不再限制最大缩放比例为 1，小网格也能完整显示

**使用方法**：
1. 上传图片后，画布会自动适应容器大小
2. 使用"放大"/"缩小"按钮调整视图
3. 点击"重置"按钮恢复到完整显示状态
4. 按住 Ctrl/Cmd + 滚轮进行精细缩放

**技术实现**：
- 移除了 `Math.min(scaleX, scaleY, 1)` 中的 `1` 限制
- 每次网格变化时自动重置缩放和位置
- 确保画布始终居中显示

### 2. 一键去杂色功能 ✅

**功能描述**：
- 根据占比阈值自动识别并移除数量少的杂色
- 支持 0.1% - 1.0% 的占比范围设置
- 自动将杂色替换为最接近的其他颜色
- 支持一键恢复操作

**使用方法**：

#### 步骤 1：设置占比阈值
- 在"一键去杂色"面板中，使用滑块调整占比阈值（默认 0.5%）
- 阈值说明：
  - 0.1%：移除数量极少的颜色（最保守）
  - 0.5%：移除数量较少的颜色（推荐）
  - 1.0%：移除数量少的颜色（最激进）

#### 步骤 2：应用去杂色
- 点击"应用去杂色"按钮
- 系统会自动：
  1. 计算总珠子数量
  2. 根据阈值确定需要移除的颜色
  3. 将这些颜色替换为最接近的其他颜色
  4. 显示移除的颜色数量

#### 步骤 3：查看结果
- 画布会实时更新显示去杂色后的效果
- 颜色统计会自动更新
- 面板会显示"已移除 X 种杂色"的提示

#### 步骤 4：恢复（可选）
- 如果对结果不满意，点击"恢复"按钮
- 画布会恢复到去杂色前的状态

**计算示例**：
```
假设：
- 总珠子数：10,000 颗
- 占比阈值：0.5%
- 阈值珠子数：10,000 × 0.5% = 50 颗

结果：
- 数量 ≤ 50 颗的颜色会被移除
- 例如：红色 45 颗、蓝色 30 颗 → 都会被移除
- 例如：绿色 60 颗 → 保留
```

**技术实现**：
- 使用欧氏距离算法找到最接近的替换颜色
- 深拷贝网格数据，确保可以恢复
- 实时计算颜色统计

**注意事项**：
1. 去杂色操作会影响整个图纸，建议先预览效果
2. 可以多次调整阈值并重新应用
3. 恢复操作只能恢复到上一次去杂色前的状态
4. 去杂色后可以继续使用手动编辑功能微调

## 功能对比

### 手动去杂色 vs 一键去杂色

| 特性 | 手动去杂色 | 一键去杂色 |
|------|-----------|-----------|
| 操作方式 | 在颜色统计中逐个点击排除 | 设置阈值一键应用 |
| 适用场景 | 精确控制要移除的颜色 | 快速移除所有杂色 |
| 灵活性 | 高（可选择任意颜色） | 中（基于数量自动判断） |
| 效率 | 低（需要逐个操作） | 高（一键完成） |
| 恢复方式 | 逐个恢复 | 一键恢复 |

**建议使用流程**：
1. 先使用"一键去杂色"快速移除大部分杂色
2. 如果还有个别不满意的颜色，使用"手动去杂色"精确调整
3. 最后使用"手动编辑"功能进行细节微调
