## 📖 使用指南

### 方式一：上传图片

1. 点击或拖拽图片到上传区域
2. 调整网格尺寸（10-300）
3. 调整颜色数量控制（0-100）
4. 点击"应用参数"
5. 可选：去除背景、手动编辑
6. 导出图纸或 CSV

### 方式二：上传 CSV

1. 点击"下载 CSV 模板"获取标准格式
2. 编辑 CSV 文件（Excel 或文本编辑器）
3. 上传 CSV 文件
4. 系统自动生成图纸
5. 可继续编辑或导出

## 📄 CSV 文件格式

### 标准格式
```csv
x,y,color,code
0,0,#FF6B9D,A01
0,1,#4ECDC4,B12
0,2,#FFE66D,C23
```

### 列说明
- `x`: 横坐标（从 0 开始）
- `y`: 纵坐标（从 0 开始）
- `color`: 十六进制颜色值（必需）
- `code`: 品牌色号（可选）

### 示例：创建 3x3 笑脸图案
```csv
x,y,color,code
0,0,#FFFFFF,W01
0,1,#000000,K01
0,2,#FFFFFF,W01
1,0,#FFFFFF,W01
1,1,#FFFFFF,W01
1,2,#FFFFFF,W01
2,0,#000000,K01
2,1,#000000,K01
2,2,#000000,K01
```

## 💡 使用技巧

### 获得最佳效果
1. 使用清晰、对比度高的图片
2. 从小网格开始（30-50）
3. 逐步调整颜色合并阈值
4. 使用"去除背景"功能简化图案

### CSV 工作流
1. 在 PC 端设计图案
2. 导出 CSV 文件
3. 在移动端继续编辑
4. 跨设备无缝协作

### 颜色管理
1. 点击颜色统计表排除杂色
2. 使用批量替换统一颜色
3. 自动去除杂色功能快速清理

## 🐛 故障排除

### CSV 上传失败
- 检查文件格式是否正确
- 确保包含 x, y, color 列
- 颜色值使用十六进制格式（#RRGGBB）

### 图片处理缓慢
- 降低网格尺寸
- 使用较小的图片
- 关闭浏览器其他标签页

### 导出失败
- 检查浏览器是否允许下载
- 尝试刷新页面重试
